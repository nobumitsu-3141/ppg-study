# PPG 波形形態からの血管トーン非侵襲推定

**SpO₂ プレチスモグラフィ波形の形態解析による、校正不要・非侵襲的な血管トーン指標の開発**

> SpO₂（**peripheral capillary oxygen saturation**：経皮的動脈血酸素飽和度）モニタが表示する脈波 ―― すなわち **PPG（photoplethysmography：光電容積脈波）** ―― の「形」から、血管の緊張状態（血管トーン）をどこまで読み取れるか。本リポジトリは、その臨床的・理論的根拠を文献にもとづいて整理し、検証したドキュメント群である。麻酔科専門医の視点、とくに **離島の一人麻酔科**という制約下での実装可能性を軸に据えている。

## 1. このプロジェクトは何か

### 中心的な問い

パルスオキシメータの脈波（PPG）は、指先の組織に光をあて、拍動に伴う**血液量の変化**を光の強さの変化としてとらえた信号である。ここで決定的に重要なのは、**PPG は「圧」ではなく「容積（血液量）」を反映する信号**だという点で、この性質が以降のすべての指標の解釈可能性と限界を決めている。

このありふれた信号の「波形の形」から、観血的なカテーテルを入れずに **血管トーン（血管の緊張・全身血管抵抗）の変化**を推定できないか ―― これが本プロジェクトの中心的な問いである。

### 設計思想 ―― 絶対値ではなく「変化（Δ）」を追う

多くの既存指標は、波形の特徴量を血圧や全身血管抵抗の**絶対値**に対応づけようとして、機器ごと・個人ごとのばらつきに阻まれてきた。本プロジェクトは方針を変え、**麻酔導入前のベースラインからの相対変化（Δ：デルタ）を追跡する**ことを設計の核心に置く。

- **校正が要らない**：個人内の変化だけを見るため、絶対値校正が不要になる。
- **計算式に依存しない**：平均動脈圧（**MAP**：mean arterial pressure）の推定式のような、機器固有の変換式に縛られない。
- **臨床的に妥当**：手術中に問題になるのは「導入前と比べて今どう変わったか」であり、Δ の追跡はこの臨床的関心と一致する。

### なぜ「一人麻酔科」の視点なのか

離島で麻酔科医が一人という状況では、高価な追加モニタを常用できるとは限らず、手元にある標準的なパルスオキシメータ（Nihon Kohden Life Scope）から最大限の情報を引き出す価値が高い。同時に、臨床モニタの表示波形は **AGC（automatic gain control：自動ゲイン制御）**と帯域制限（バンドリミティング）を受けており、研究用光電計での知見をそのまま持ち込めない。この「実機の制約」を各ドキュメントで正面から扱っている点が、本プロジェクトの特徴である。

## 3. 各成果物の詳細

### 3.1 中核レビュー：SpO₂/PPG 波形解析の流れ（`SpO2_PPG_waveform_analysis_flow.html` / `.md`）

本体となる文献レビュー。デザインシステムに沿った **HTML 版**（閲覧用）と、編集可能な **Markdown ソース**（`.md`、GitHub 上でそのまま描画される）の両方を同梱している。パルスオキシメータの脈波を 8 つの節で整理している。

- **§1 光電容積脈波としての成り立ち**：PPG が圧ではなく容積信号であること、直流成分（**DC**）・交流成分（**AC**）・灌流指数（**PI**：perfusion index＝AC/DC）の定義。
- **§2 前進波と反射波**：波形を読むための土台。左室駆出による**前進波**と、末梢で反射して戻る**反射波**の合成として波形を理解する枠組み。重複切痕を「大動脈弁閉鎖」だけに帰属させる古典的説明の限界を示す。
- **§3 循環因子の変化に伴う波形変化**：前負荷・全身血管抵抗・血管壁スティフネス・心収縮力・心拍数の各因子について、「循環の変化 → 波形の変化」を対応づけ、**PPG での検証の強さ（実測か外挿か）を併記**。
- **§4 目視による定性的評価**：Murray & Foster（1996）に始まる「切痕は血管トーヌスを映す」という発想と、その限界（トレンド読解には有効だが絶対評価には不向き）。
- **§5 波形変化の定量化**：**SI（stiffness index：スティフネス指数）**・**RI（reflection index：反射指数）**・**AIx（augmentation index：オーグメンテーション指数）**・加速度脈波などを、血管特性の軸と急性トーヌスの軸に分けて整理。
- **§6 高スティフネス血管の切痕検証**：スティフネスが上がると切痕は「むしろ消える」というパラドクスを、消失の程度と二次微分（**SDPTG**：second derivative of photoplethysmogram＝加速度脈波）の c–d 波で解消する。

### 3.2 変曲点を血管抵抗として読む（`PPG_dicrotic_notch_resistance_review.html`）

「重複切痕の位置が血管抵抗を映す」という考えの**研究史レビュー**。19世紀の脈波計法による切痕の発見から、Takazawa ら（1998）の加速度脈波、Chowienczyk ら（1999）の反射指数（＝小血管トーヌス）、Chételat/Sola ら（2018）による周術期6クラス分類、Pal ら（2024）の自動検出アルゴリズム、進行中の臨床試験（NCT04179097）までを、一次文献の URL つきで追う。

### 3.3 LVET と 6 因子の波形図解（`PPG_LVET_six_factors_figures.html`）

6つの循環因子（前負荷・心収縮力・血管スティフネス／PWV・全身血管抵抗・特性インピーダンス Zc・血流の慣性 L）が、波形の**どの領域に・どの向きに**効くかを、領域マップと各因子ごとのスパークライン（基準波形と変化後を重ねた小さな波形図）で視覚化した図解ドキュメント。

### 3.4 6 因子の検証プロベナンス・第4版（`PPG_fig2_factor_references_4_2.html`）

上記の図解が示す各矢印（因子↑で波形がこう動く）について、その根拠が

- **実測 PPG あり**（パルスオキシメータ波形を観血的動脈圧や生理変化と直接照合した研究がある）
- **現象は実測・機序はモデル**（波形変化そのものは観察されるが、原因の帰属はモデル依存）
- **動脈圧外挿・モデル**（観血的動脈圧の理論や in silico モデルからの外挿で、実測 PPG での検証がない）

の 3 区分のどれに当たるかを監査した文書。結論として、**実測 PPG で直接裏づけられるのは前負荷と全身血管抵抗の 2 因子のみ**（いずれも条件付き）であることを明示し、エビデンスの強さを均一に扱う過剰主張を戒める。

### 3.5 T₂/PI インタラクティブツール（`t2_pi_normalization.html`）

血管緊張が変化したときに、合成指標（脈波伝播時間の遠位成分 **T₂** と灌流指数 **PI** の比）がどう振る舞うかを、スライダーで操作しながら確かめられる可視化ツール。単調でない（U字型／回復する）挙動を、関数形式の理論的正当性とあわせて示す。

### 3.6 PWTT と esCCO ― 時間区間としての脈波（`PWTT_esCCO_structure_and_limits.html` / `.md`）

既存の各成果物が扱う波形の**形態**とは直交する軸、すなわち脈波の**時間区間**を扱う技術レビュー。モニタが表示する **PWTT（pulse wave transit time：脈波伝播時間）**の正体は、心臓側の **PEP（pre-ejection period：前駆出期）**と血管側の **VTT（vascular transit time：血管内伝播時間）**の和である **PAT（pulse arrival time：脈波到達時間）**であることを構造から説明する。**esCCO** の較正係数 K に血管情報が入っていないために、後負荷上昇時に一回拍出量を系統的に過大評価する機序、誤差率およそ 47%（Critchley 基準 ±30% を超える）という精度の限界、そして限界の根源が「大動脈弁開放を捉えていない」一点にあることを、複数の実測研究の数値とともに示す。既存 §5 の「esCCO の全身血管抵抗依存性バイアス」の一行に対応する本体レビューである。

### 3.7 esCCO の後負荷交絡を波形形態で補正する（`esCCO_K_correction_hypothesis.html` / `.md`）

**未検証の作業仮説を、反証可能な形に整えて提示する文書**（他のレビューと視覚的に区別している）。esCCO の較正係数 K を定数ではなく、PWTT の算出に用いるのと同一の光電容積脈波から得られる加速度脈波の形態指標（d/a・e/a など）の関数として**動的に補正する**という提案を、段階 0〜3 の検証設計とともに示す。成立の必要条件は、臨床モニタの帯域制限下でも d/a・e/a が計算できることであり、これが否定されれば提案は成立しない。新規性の主張は暫定的であり、系統的な先行研究・特許検索によって確定させる必要があることを明記している。

### 3.8 加速度脈波の予後エビデンスと拡張期側の空白（`SDPPG_prognostic_evidence_and_diastolic_gap.html` / `.md`）

**SDPPG（加速度脈波）**の b/a・d/a 比が実際に何を予測したか——頸動脈伸展性・内膜中膜厚、Framingham リスクスコア、そして中高年女性 4,373 名を追跡した前向きコホートでの心血管死——という**予後エビデンス**を整理する（既存文書は指標の紹介にとどまっていた）。あわせて 2024–2026 年の携帯デバイスへの実装動向と、原波形の拡張期ピークに対応する「**f 点**」が SDPPG 上でまだ操作的定義も検証済み指標も持たないという空白を扱う。第 4 節（f 点仮説）は検証されていない仮説の提示であり、視覚的に区別している。

### 3.10 周術期アウトカムと血管スティフネス（`perioperative_stiffness_outcomes.html` / `.md`）

術前の血管スティフネスが麻酔導入時低血圧や術後合併症を予測するかを問う臨床アウトカムレビュー。核心は個々の研究結果ではなく、それらを並べたときに見える一つの不都合な構造——**取得が容易な指標（SpO₂ センサ由来）ほど周術期エビデンスが乏しく、確かなエビデンスがある指標（cfPWV・baPWV など専用機を要するもの）ほど現場で取れない**——にある。PPG 由来の動脈スティフネス指数 ASI が 12.7 万人規模のコホートで心血管イベントを予測する一方、その予測力が最も血管の硬い高齢者で消えることも示す。通し番号 9「心拍数は PPG 波形の『形』を変えるか」は準備中で未収録である。

## 4. 研究の設計原則

各ドキュメントを貫く方法論上の原則。

### 4.1 絶対値ではなく個人内 Δ を追う

前述のとおり、非校正の PPG 指標が臨床的に妥当でありうる基盤は、**個人内ベースラインからの相対変化（Δ）追跡**にある。AIx についても、心拍数補正（75 bpm への正規化＝AIx@75）の必要を認識したうえで、Δ 追跡は妥当である。

### 4.2 タイミング／形状の情報を、振幅より優先する

臨床モニタの **AGC（自動ゲイン制御）**は、振幅の絶対値や拍間の振幅変化（呼吸性変動 ΔPOP など）を破壊する。一方で、**同一波形内の振幅比**（b/a, d/a, RI）や**タイミング指標**（SI, 立ち上がり時間）は AGC 後も保持される。したがって指標設計では、振幅比よりタイミング／形状の位置情報を優先するのが安全である。

- 補足：加速度脈波（SDPTG）の形態特徴（c 波・d 波）への主要な脅威は、AGC ではなく**バンドパスフィルタリング（帯域制限）**である。この区別は実装上きわめて重要。

### 4.3 脈波伝播速度（PWV）は「原因」ではなく「結果の指標」

血管壁スティフネスの節では、**PWV（pulse wave velocity：脈波伝播速度）**を因子そのものとして扱わない。PWV は壁が硬いこと（スティフネス）の**結果を測る指標**であって、原因因子ではない。この位置づけは全ドキュメントに適用される原則である。

### 4.4 エビデンスの強さを均一に扱わない

「Aライン（観血的動脈圧）で言えること」と「実測 PPG で確認されたこと」を厳密に区別する。多くの教科書的記述は前者からの外挿であり、PPG で同じ挙動が確認されているとは限らない。この確信度の差を、各主張に明示的に付す。

## 5. これまでに確立した主要な知見

| 論点 | 到達点 |
|------|--------|
| **引用の厳格な検証** | 過去に著者名の取り違えが生じた反省から、すべての臨床・研究的主張に、NCBI E-utilities または Web 検索で検証済みの PMID／DOI／URL を付す運用を徹底 |
| **AGC と帯域制限の区別** | AGC は振幅の絶対値・拍間変化を壊すが、波形内振幅比・タイミング指標は保持。SDPTG 形態への主脅威は帯域制限であり AGC ではない |
| **PWV の位置づけ** | PWV はスティフネスの結果を測る指標であり、原因因子ではない（文書全体の原則） |
| **esCCO の全身血管抵抗依存性バイアス** | 脈波伝播時間（**PWTT**）の短縮は 1回拍出量（**SV**）増加でも全身血管抵抗増加でも起こるが、後者では SV が同時に低下するため、全身血管抵抗のみが上昇する場面で **esCCO は SV を系統的に過大評価**する。トーン信号をこのノイズから分離するには PWTT 遠位成分 T₂ の単離が核心となる（構造と実測データの詳細は成果物 6） |
| **PWTT の正体は PAT である** | モニタ表示の PWTT は脈波到達時間（**PAT**）であり、前駆出期 PEP と血管内伝播時間 VTT の和。PEP を分離できないことが esCCO の全限界の根源である（成果物 6） |
| **心拍数交絡は指標依存である** | 反射指数 RI は心拍数と関連するが、スティフネス指数 SI は関連が乏しく、加速度脈波指標は結論保留。「心拍数上昇で反射波が高くなる」という記述は**支持されない**（成果物 10 §6.2） |
| **取得容易性とエビデンスの逆相関** | 周術期において、SpO₂ センサだけで取れる指標ほど検証が乏しく、専用機を要する指標ほどアウトカム研究がある（成果物 10） |
| **相対変化追跡の妥当性** | 絶対値校正ではなく個人内 Δ 追跡が、非校正 PPG 指標の臨床的妥当性の基盤 |
| **実測 PPG の裏づけがある因子** | 6 因子のうち、実測 PPG で直接裏づけられるのは前負荷と全身血管抵抗の 2 つのみ（条件付き） |

> **esCCO とは**：estimated continuous cardiac output（推定連続心拍出量）。Nihon Kohden が実装する、心電図と脈波の時間差（脈波伝播時間）から連続的に心拍出量を推定する方式。
