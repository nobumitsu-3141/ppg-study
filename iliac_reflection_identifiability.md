# 腸骨反射（P3）の識別性を高めた PPG 波形分解モデル

> 指尖 PPG から反射波の**位置（時相）と大きさ**を推定するとき、拡張期側＝**腸骨（下大動脈以遠）反射**を
> 標的成分にする（別稿 `reflection_wave_position_RI_afterload.md` の議論の続き）。本稿は、その腸骨反射を
> **識別可能（identifiable）に推定するための分解モデル**を定義し、合成波（真値既知）で各対策の効き目を
> 実測した記録である。検証コードは同梱 `iliac_reflection_identifiability.py`（`python3` で再現可能）。

---

## 1. なぜ腸骨反射を標的にするのか

反射の到達時刻 ＝ 経路長 ÷ PWV。**腸骨（最遠位）反射は最も経路が長く、血管の硬さ（PWV）で最もよく前後する
"可動成分"**である：柔らかい血管では拡張期ピーク、硬い血管では後期収縮期へ前進して後負荷（late-systolic
load）に化ける。したがって腸骨反射の**時相 τ_iliac** を追えば、後負荷が収縮期に載ったか（＝左室後負荷）と
大血管スティフネス（PWV）の両方を捉えられる。振幅 Γ_iliac は遠位トーヌス（SVR）を映すが、PPG では容積≠圧で
歪むため二次指標とする。

問題は、この腸骨成分が**最も見えにくい**点にある：硬いと収縮期へ融合して独立ピークが消え、柔+速いと次拍に
呑まれる。だから「ピーク検出」では非頑健で、**モデル分解（フィット）で潜在成分として推定**する必要がある。
本稿はそのフィットを**識別可能**にする。

---

## 2. モデル定義

1 拍の PPG を `y(t)`（容積、baseline 除去済み）とし、前進波と2つの反射（腎・腸骨）の重ね合わせで表す。

```
y(t) = W_f(t) + Σ_{k∈{renal, iliac}} Γ_k · W_f( (t − τ_k) / β ) + ε(t)
W_f(t) = A · s(t; μ, σ_r, σ_f)        s = 単峰・非対称パルス（rise σ_r / fall σ_f）
```

- **反射は前進波の "遅延 τ_k・縮尺 Γ_k・分散 β" コピー**（＝ shape-tying）。独立パルスにしない。
- 推定パラメータは **8 個**：`θ = [A, μ, σ_r, σ_f, Γ_renal, τ_renal, Γ_iliac, τ_iliac]`（β は分散、まず固定）。
- 基底 `s` は **hyperbolic secant（sech²）または lognormal** を推奨（Moens–Korteweg 進行波に整合し、反射
  パラメータの拍間ノイズが小さい）。本検証では非対称ガウスを用いた。
- 目的関数：`Σ_t ((ŷ−y)/σ)²  +  (τ_iliac − τ̂)²/σ_prior²  +  λ·(拍間平滑)`、box 拘束つき。

> **設計の要点** ―― 反射項の "形" を前進波に縛る（shape-tying）ことが識別性の本体である。反射を自由な
> パルスとして置くと、位置・幅・振幅が互いにトレードオフして腸骨の位置が定まらない（下記実験1）。

---

## 3. 識別性の対策（合成波で検証）

真値既知の合成波で、腸骨ピーク時刻（μ+τ_iliac）の推定誤差と Cramér–Rao 下限（CRLB）を測った。数値は
`iliac_reflection_identifiability.py`（seed 20260721）の出力。

### 3.1 形状拘束（shape-tying）＝ 主レバー

自由3ガウス（12 パラメータ）と形状拘束（8 パラメータ）を比較。

| シナリオ | 指標 | 自由3ガウス | 形状拘束 |
|---|---|---|---|
| A 明瞭な拡張期ピーク（真 480 ms, noise 0.04） | CRLB | 6.0 ms | **2.5 ms** |
| 〃 | RMSE | 4.2 ms | **2.2 ms** |
| B 硬い/融合/弱い腸骨（真 350 ms, 最難） | **CRLB** | **28.9 ms** | **6.9 ms** |
| 〃 | RMSE | 22.8 ms | 21.2 ms |

**融合した最難ケースで、CRLB（局所情報限界）が ±29 ms → ±7 ms と 4 倍改善**。ただし B の RMSE は拘束でも
21 ms と大きい ―― これは局所限界ではなく**時折の誤収束（悪い局所解）**が支配するため。これを潰すのが次の
アンカーである。

### 3.2 生理拘束・順序

- `τ_renal < τ_iliac`（ラベル入替＝腎↔腸骨の取り違え防止）。
- Baruch の窓：`τ_renal ∈ [70,140] ms`、`τ_iliac ∈ [180,450] ms`。`Γ_k ∈ [0, 0.9]`。

### 3.3 身長/PWV/SDPPG アンカー（＝ 頑健性、精度ではない）

`τ_iliac` に軟らかい事前分布（`τ̂ ≈ 2·c·身長/PWV`、σ_prior≈30 ms、SDPPG の f 点でも可）を与える。硬い/融合
regime・高ノイズ（0.06, N=250）：

| | MAE | RMSE | gross-fail(>40 ms) |
|---|---|---|---|
| アンカー無し | 13.4 ms | 27.5 ms | **7.2 %** |
| アンカー有り(30 ms) | 9.0 ms | **12.2 ms** | **1.2 %** |

> **正直な訂正** ―― 以前「アンカーで識別性（CRLB）が上がる」と述べたが、検証では**局所 CRLB はほぼ不変**で、
> アンカーが効くのは**"誤った局所解に落ちる"のを防ぐ頑健性**だった（gross-fail 7.2→1.2 %、RMSE 27.5→12.2 ms）。
> 識別性の本体は 3.1 の形状拘束であり、アンカーはそれを実運用で"取りこぼさない"ための保険である。

### 3.4 退化ケース

- **硬い/融合（独立ピーク消失）**：SDPPG の c/d/f 変曲で初期化し、肩から τ_iliac を推定（峰不要）。
- **wrap-around（柔+速、腸骨ピークが次拍 foot より後ろ）**：真 750 ms > 周期 700 ms の合成で、
  1拍窓（切断）＝ **746.2 ms**、2拍窓（裾を回収）＝ **750.5 ms**。形状拘束モデルは切断されても立ち上がりから
  外挿して大きくは外さない（残差 −4 ms）が、**2拍窓＋次拍 foot 境界**で残差が消える。安全策として推奨。

## 4. 識別性診断（拍ごとに算出し信頼度に使う）

- **FIM → CRLB(τ_iliac)**：小さければ採用、大きければ低信頼としてトレンドで重み下げ。
- **プロファイル尤度(τ_iliac)**：明瞭な最小か、平坦な尾根（＝非識別）か。
- **ヤコビアン条件数／(Γ_iliac, τ_iliac) と前進波裾の相関**：高相関＝非識別。3.1・3.2 で解消。

---

## 5. 推定パイプライン

1. 拍分割（foot 検出）→ baseline 除去 → **振幅正規化**（AGC 対策：比・時相を残す）。
2. **SDPPG ランドマーク**（a–e, f）で初期値（μ, τ_renal, τ_iliac）。
3. 拘束付き非線形最小二乗（TRF）＋ τ_iliac アンカー。
4. 拍ごと CRLB → 信頼度フラグ。
5. **多拍の状態空間平滑**（τ_iliac, Γ_iliac を緩いランダムウォーク、拍 CRLB を観測ノイズ）でトレンドを安定化。
6. 出力：**τ_iliac / T13（時相＝PWV・後負荷が収縮期に載ったか）を一次指標**、**Γ_iliac（遠位トーヌス、
   PPG では歪む）を二次**。

---

## 6. 限界と検証設計

- **「反射」ラベルはモデル仮定**：reservoir（windkessel）・再反射・上半身・腕/指局所を反射項に吸収する。
  → 観血/中心圧を参照に対応づける検証が要る。
- **β（分散）固定は近似**。硬さで dispersion が変わるなら τ・Γ に偏りが乗る → β も緩く推定 or 硬さ層別。
- **アンカーは頑健性用（精度ではない）**。真に埋もれた腸骨では形状拘束でも限界 → 低信頼拍は捨てる。
- **合成波での検証**である。実 PPG（VitalDB など）＋参照基準（観血・中心圧・熱希釈 SVR/Ea）での**前向き
  検証が必須**。
