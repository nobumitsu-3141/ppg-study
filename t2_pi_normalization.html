PPG／PWTT 波形解析 · 血管トーン指標

T₂ / PI の挙動

単調減少する PI と U字型の T₂ を合成したとき、正規化指標がどう振る舞うかのシミュレーション。相対値（術前ベースライン = 1）で表示。

01モデルの定義

x ＝ 正規化血管緊張（0 = 術前ベースライン、1 = 極度収縮）。以下の3量はすべてベースライン値 = 1 に正規化。 α = PI の指数減衰率、δ = T₂ ディップ深さ、x* = T₂ 最小点の位置。

PI(x) = e−αx
T₂(x) = 1 − (2δ/x*)·x + (δ/x*2)·x2   （最小点 x*、最小値 1−δ の二次関数）
T₂/PI(x) = T₂(x) · eαx   （分母が高トーン域で急減 → 増幅）

03なぜこうなるのか

T₂ が U字になりうる機序（下降 → 上昇）

下降脚（中等度収縮）：トーン亢進と近位圧上昇で壁が硬化しコラーゲンが係合、筋性動脈の局所 PWV が上昇して T₂ は短縮する。

上昇脚（極度収縮）：抵抗血管を挟んだ遠位で経壁圧が崩れ、エラスチン優位のコンプライアント域に移る。PWV の圧依存性により局所 PWV が低下し、T₂ は延長へ転じる。波の高度減衰による foot 検出遅延も加わる。

PI で割ると単調化する理由

T₂/PI = T₂(x)·eαx。指数因子 eαx は単調増加で、T₂ の浅いディップを覆い隠す。全域で単調増加となる条件は次のとおり。

α > 2δ/x*

生理的に PI は 10〜20 倍規模で低下（α 大）するため通常は満たされる。結果、T₂ 単独の「1つの値が2つのトーン状態に対応する」多義性が解消され、高トーン域で高感度な一価指標になる。

但し書き

PI はトーン単独ではなく灌流圧・SV・トーンの合成物。昇圧薬投与時は灌流圧・SV も上がるため PI が上昇しうる。本モデルは「灌流圧が概ね一定〜低下する文脈」を前提とする。また T₂ の U字自体は機序から演繹した仮説であり、直接測定した報告は現時点で確認できていない。

実務メモ ― 一人麻酔科の視点

ベッドサイドでの含意

手元の Life Scope で連続取得できる PI と、PWTT（esCCO）を組み合わせれば、追加デバイスなしに T₂/PI に相当する比を術前 Δ ベースで追える可能性がある。T₂ 単独では「中間値が緩和と極度収敏のどちらか判別できない」場面で、PI 正規化はその曖昧さを取り除く。

特に高トーン域での急峻な立ち上がりは、寒冷・出血・深麻酔からの覚醒などで末梢が締まる局面の早期検出に向く。ただし PI が昇圧薬で逆方向に動く点は必ず併せ読みすること ― NA 開始直後の PI 上昇を「トーン低下」と誤読しないよう、投薬イベントと同期して解釈する。
