## 0. 川副式デザイン規約（全スライド共通・厳守）

- 右上パンくず（8章）：現在章＝ティール `#00A8AA`＋白、他＝薄グレー。ラベルは各2字：**歴史 / 原理 / 生理 / 正常 / 術中 / 異常 / 対応 / 特殊**。
- 配色パレット（これだけ使う）：ゴールド`#BF9000`（主役・キー用語/数式）、ティール`#00A8AA`（現在地・アクセント）、インク`#262626`（本文・波形線）、グレー`#808080`（補足）、青`#1F4E79`/地`#DDEBF7`、橙`#8A3B00`/地`#FBE5D6`、緑`#70AD47`/地`#C5E0B4`、赤`#C00000`（危険・注意）。
- 文字最低サイズ：目次/章扉見出し≥28pt、本文≥22pt。注釈（出典10.5pt・ページ番号12pt・図内ラベル）は対象外。
- **単語の途中で改行しない。文字・図・記号（矢印/×/＝/≈）を重ねない。**
- 出典＝左下グレー10.5pt・1行、**一次文献にハイパーリンク**。形式 `著者. 誌名 年; 巻:頁`。
- ページ番号＝右下 `N/総数`（表紙は付けない）。

## 1. スライド種別（レイアウト型）

- **title**：表紙。大タイトル＋サブ＋氏名＋下部ヒーロー波形。
- **cq**：クリニカルクエスチョン（臨床の問いで掴む・ダジャレ可）。
- **goals**：到達目標3〜4点。
- **menu**：8章見出しを大きく行間広め。
- **divider**：章扉。全8章を縦列、現在章のみゴールド＋節番号（黒）、他薄グレー。
- **figure**：金タイトル＋下線＋模式図（主役）＋出典。テキスト最小。← 主力。
- **bullets**：箇条書き（要点整理）。
- **column**：淡色角丸ボックスに見出し＋2〜3文（用語整理・限界・実務メモ）。
- **table**：パターン→原因→対応 等の一覧（figureとして描くか、テキスト2段組）。

## 2. Opus への執筆依頼（出力フォーマット）

各スライドについて、以下を JSON 風の構造化テキストで返す（ビルドにそのまま渡せる粒度）：

```
- id: 章.節連番（例 4.1a）
  title: 金タイトル（1行・短く。節番号込み）
  layout: figure | bullets | column | table
  body:
    # bullets型: 箇条書き（各行1行に収まる長さ、22pt想定）
    # column型: heading（見出し1行）＋ paras（2〜3文）
    # figure型: caption（任意・1行）
  figure_spec:            # figure/table型で、matplotlibで描く模式図の設計
    what: 何を描くか（波形/軸/ボックス/矢印の配置を言葉で）
    labels: 図中ラベル（ゴールド=キー、青/橙/緑=分類、赤=危険）
    values: 図中に入れる数値・角度・式（正確に）
  citation: "著者. 誌名 年; 巻:頁"（一次文献）
  cite_url: PubMed/DOI 等の一次リンク
  clinical_notes: 監督(Fable)が裏取り確認すべき点／自信のない数値は [要確認] を付す
```

**執筆方針（厳守）:**
- 麻酔科専門医向けに**臨床運用の視点**で。生理の式で足場を作る（例 `V̇CO₂ ∝ 代謝`、`PaCO₂ = k·V̇CO₂/V̇A`、a–ET 較差）。
- **著作権セーフ＝完全オリジナル**。教科書・ガイドライン本文の丸写し禁止。論点・数値は参照可、文章は自作。
- **数値・カットオフ・角度・正常値は正確に**。標準的な値を書き、異論のある値は幅で示すか [要確認]。
- 各臨床主張に**一次文献**を1つ添える（可能な範囲で PubMed PMID / DOI）。分からなければ `cite_url: null` とし clinical_notes に「出典要確認」。
- 図は「ひと目で分かる模式図」。テキストで説明しきらず、figure_spec に絵の設計を書く。

## 3. デック骨格（全スライド）

### 前付け（Fable が下書き、Opus は医学的な文言のみ微修正可）
- **T. title** — 「EtCO₂ モニタリング — カプノグラフィを読み解く」／サブ「歴史・原理から術中の見どころ・特殊使用まで（麻酔科専門医向け）」／川副 靖晃／ヒーロー＝正常カプノグラム波形。
- **CQ. cq** — 「その"四角い波形"、本当に読めていますか？」毎日見る EtCO₂ 波形。相IIIの傾き・高さ・ベースラインから何が読めるか。→ 歴史→原理→生理→正常→術中→異常→対応→特殊 で体系化する。
- **G. goals** — ①測定原理（赤外吸収・main/side）を説明できる ②正常カプノグラム（4相・α/β角・a–ET較差）を読める ③異常波形をベースライン/高さ/形/リズムで系統的に鑑別できる ④EtCO₂ 変化への術中対応を選べる ⑤特殊使用（CPR・挿管確認・鎮静・腹腔鏡）を活用できる。
- **M. menu** — 8章見出し。

### 第1章 歴史（breadcrumb=0「歴史」）
- **1.0 divider**
- **1.1 figure** CO₂ の発見から気体分析へ：Black（CO₂ 同定, 1750s）→ Tyndall（赤外吸収の発見, 1859-64）→ Luft（非分散赤外ガス分析計 NDIR, 1943）。時系列の年表模式図。
- **1.2 figure** カプノグラフィの臨床導入：質量分析の時代、Smalhout & Kalenda（1970s, カプノグラム波形学のアトラス）、サイド/メインストリームの登場。
- **1.3 figure/bullets** 標準モニタとしての確立：ASA 基本モニタリング基準（1986）、Harvard 基準、食道挿管の検出＝麻酔関連死の低減。JSA モニタリング指針。
- **1.4 column** なぜ EtCO₂ は"呼吸のモニタ"の主役になったか（SpO₂ より早く換気異常を捉える／換気・循環・代謝を1つの波形で）。

### 第2章 成り立ち＝測定原理（breadcrumb=1「原理」）
- **2.0 divider**
- **2.1 figure** 赤外吸収の原理：CO₂ は約 4.26 µm の赤外を吸収（Beer–Lambert 則）。光源→サンプルセル→検出器の模式図＋吸収スペクトル。
- **2.2 figure** メインストリーム vs サイドストリーム：気道内キュベット（速い・重い・死腔）vs サンプリングライン（50–200 mL/min・遅延・水トラップ）。並置比較図。
- **2.3 figure/bullets** サンプリング・応答時間・遅延：rise time、transport delay、混合による波形なまり、高RR/低TVでの過小評価。BTPS 補正。
- **2.4 column** 誤差要因：水滴・分泌物、N₂O/O₂ による衝突広がり（collision broadening）、リーク、較正。数値・単位に注意。
- **2.5 figure** カプノメータ(数値) vs カプノグラフ(波形)／時間軸 vs 容量軸の2系統（次章・4章への布石）。

### 第3章 基礎＝CO₂ の生理（breadcrumb=2「生理」）
- **3.0 divider**
- **3.1 figure** CO₂ の産生・運搬・排泄：組織代謝(V̇CO₂)→静脈血→肺→肺胞換気(V̇A)で排出。3段の流れ図＋`PaCO₂ = k·V̇CO₂/V̇A`。
- **3.2 figure** PaCO₂ と PetCO₂：肺胞気 PACO₂ ≈ PetCO₂。理想肺での関係と、実際にズレる理由への橋渡し。
- **3.3 figure** a–ET(CO₂) 較差（勾配）：正常 2–5 mmHg。死腔・V/Q 不均等・拡散で規定。図で PaCO₂＞PACO₂＞PetCO₂ の階段。
- **3.4 figure** 死腔と V/Q：解剖学的/肺胞/生理学的死腔、West zone、Bohr–Enghoff。死腔↑で PetCO₂↓・較差↑。
- **3.5 figure** EtCO₂ を動かす3要素：**産生(代謝)・運搬(循環/肺血流)・排出(換気)**。以後すべての異常を読む"3軸"フレーム（この図はデックの背骨）。

### 第4章 正常カプノグラム（breadcrumb=3「正常」）
- **4.0 divider**
- **4.1 figure** 時間軸カプノグラムの4相：相I(死腔・CO₂≈0)→相II(急上昇・気道と肺胞気の混合)→相III(肺胞プラトー)→相0/吸気下降。★主役図。各相を色分け。
- **4.2 figure** α角・β角：α角(II–III 移行, 約100–110°)、β角(III–吸気, 約90°)。角の増大が示す病態への布石。
- **4.3 figure** 相IIIの傾き（勾配）の意味：軽い上り勾配は正常、勾配増＝V/Q 不均等・呼出遅延。
- **4.4 figure** 容量軸カプノグラム（volumetric capnography）：SBT-CO₂ の相I/II/III、Fowler 解剖学的死腔、Bohr 死腔、曲線下面積＝V̇CO₂。専門的深掘り。
- **4.5 bullets/column** 正常波形チェックリスト：①ベースライン0 ②矩形(急峻な立上り・平らなプラトー) ③高さ PetCO₂ 35–45 mmHg ④α角正常。4点を毎回確認。

### 第5章 麻酔中の見どころ（breadcrumb=4「術中」）
- **5.0 divider**
- **5.1 figure** 気管挿管の確認＝ゴールドスタンダード：持続する矩形波6呼吸＝気管、平坦＝食道（数呼吸で消失）。
- **5.2 figure** 換気の適切さ・回路監視：低換気/過換気、回路外れ・閉塞の即時検出（波形消失）。SpO₂ より早い。
- **5.3 figure** 循環の窓：EtCO₂ と心拍出量／肺血流。突然の低下＝塞栓・低心拍出・心停止。トレンドで循環を読む。
- **5.4 figure** 代謝・体温での上昇：悪性高熱(MH, 早期・鋭敏な指標として EtCO₂ 上昇)、敗血症、駆血解除、腹腔鏡 CO₂ 吸収。
- **5.5 figure** トレンドで読む：自発呼吸の出現（curare cleft）、麻酔深度・鎮痛、人工呼吸設定の最適化。
- **5.6 column** 術中の a–ET 較差の変化：体位・PEEP・低心拍出・肺塞栓で較差拡大。PetCO₂ だけで PaCO₂ を過信しない。

### 第6章 異常所見（波形パターン・アトラス）（breadcrumb=5「異常」）
- **6.0 divider**
- **6.1 figure** 異常の読み方＝系統的アプローチ：**ベースライン・高さ(振幅)・形・呼吸数/リズム** の4視点。読影の型。
- **6.2 figure** ベースライン上昇（吸気 CO₂ ＞0）＝リブリージング：吸収剤消耗、弁不良、新鮮ガス不足。
- **6.3 figure** 相III上昇勾配・シャークフィン＝閉塞性（気管支攣縮・COPD）。α角増大・プラトー消失。
- **6.4 figure** カーテンクレフト（curare cleft）：調節換気中の自発呼吸のくぼみ。筋弛緩の醒め。
- **6.5 figure** EtCO₂ 急低下／波形消失：回路外れ・完全閉塞・無呼吸 vs 心停止・肺塞栓・サンプリング異常の鑑別（形の違い）。
- **6.6 figure** 漸増 vs 漸減トレンド：漸増＝低換気/CO₂吸収/体温上昇/代謝亢進、漸減＝過換気/心拍出低下/体温低下。
- **6.7 figure** 特徴的パターン集：心原性振動(cardiogenic oscillation)、二相性プラトー(片肺移植等)、リーク波形、"tails"。ミニアトラス。

### 第7章 対応（トラブルシューティング）（breadcrumb=6「対応」）
- **7.0 divider**
- **7.1 figure** EtCO₂ 消失への対応アルゴリズム：患者か・チューブか・回路か・モニタか。DOPES 的な系統確認フロー。
- **7.2 figure** 急低下の鑑別と対応：塞栓／低心拍出／心停止。触診・血圧・EtCO₂ を統合して即応。
- **7.3 figure** 上昇への対応：まず換気(分時換気量)調整→原因検索。MH を疑う所見と初動（トリガー中止・ダントロレン準備）。
- **7.4 figure** 閉塞性パターンへの対応：麻酔深度・気管支拡張薬、シャークフィン改善の確認。
- **7.5 column** リブリージング・回路異常への対応：吸収剤交換・弁点検・新鮮ガス増量。
- **7.6 table** トラブルシューティング総覧：**波形/所見 → 主な原因 → 初動** の一覧表（デックの実務コア）。

### 第8章 特殊な使用例（breadcrumb=7「特殊」）
- **8.0 divider**
- **8.1 figure** 心肺蘇生（CPR）：胸骨圧迫の質の指標、EtCO₂ <10 mmHg は予後不良/圧迫不十分、ROSC で急峻に上昇、予後予測。JRC/AHA/ERC の位置づけ。★
- **8.2 figure** 挿管・チューブ位置確認（全領域）：手術室・救急・病院前・搬送。持続波形＝気管。
- **8.3 figure** 鎮静・非挿管モニタリング：手技鎮静(PSA)、マイクロストリーム鼻カニュラ、無呼吸を SpO₂ 低下より早く検出。
- **8.4 figure** 肺塞栓・循環評価：a–ET 較差の急拡大、容量カプノグラフィで肺胞死腔(Vd/Vt)。スクリーニングの補助。
- **8.5 figure** 腹腔鏡・特殊体位・ロボット：CO₂ 気腹の吸収、皮下気腫、頭低位/気腹での換気・EtCO₂ 変化。
- **8.6 column** 挿管困難・気道管理・NIV・搬送：DAS の位置づけ、換気確認、搬送時の必須モニタ。
- **8.7 figure/bullets** 小児・新生児の留意点：小さい一回換気量、サイドストリーム希釈、マイクロストリーム、正常値の考え方。
