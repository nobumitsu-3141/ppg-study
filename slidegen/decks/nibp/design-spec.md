# デザイン仕様 — マンシェット血圧スライド（インク＆ゴールド体系 v3.13 準拠）

参照元: `SpO2(PPG)波形解析の流れ3.13.pptx` から抽出。**この実物デザインに一字一句合わせる**こと。
レンダリング見本（スクラッチパッド）: `ppg_a-01.png`(表紙) / `ppg_a-02.png`(目次) / `ppg_a-03.png`(章オープナー) / `ppg_a-04.png`,`ppg_a-05.png`,`ppg_b-12.png`(内容ページ)。

## キャンバス
- 16:9 ワイド = 13.33 × 7.5 in（12192000 × 6858000 EMU）
- 背景: 全ページ白 `FFFFFF`（現行版にダーク全面章扉は**無い**。章扉は「白背景の節目次」スタイル）
- フォント: **Meiryo** を全テキストに指定（日本語・英数字とも）

## タイプスケール（pt, Meiryo）
- 大タイトル（章オープナー「N. 章名」／内容「N.M タイトル」）: **46**、色 GOLD `BF9000`
- 目次「メニュー」見出し: 26–28、GOLD、字間広め
- 節リスト見出し（章オープナー内の各章行）: 22–24
- 節番号ラベル（N.M）: 15–18、GOLD
- 本文ブレット: 14–15、INK
- 図キャプション・小ラベル: 12
- 出典・ページ番号（フッター）: 9.5–10.5、MUTED

## コンポーネント
1. **表紙**: 白。中央やや下にブロック。大タイトル INK 濃色（`262626` 相当、太め）／サブタイトルは GOLD の emダッシュ `— … —`／三次行 MUTED／著者名 GOLD。最下部中央に小さな金の脈波モチーフ（本デッキではカフ/圧計の細線モチーフに置換可）。
2. **目次（メニュー）**: 右上にナビ（全タブ非活性グレー）。GOLD 見出し「メニュー」。GOLD 全幅罫線。1–8 の番号付きリスト（番号 GOLD・本文 INK）、行間ゆったり。
3. **章オープナー（節目次）**: 右上ナビ＝当該章を TEAL。大 GOLD「N. 章名」＋GOLD罫線。下に全章リスト。**現在章の行は INK＋その節（N.1, N.2…）をインデントして GOLD番号＋INKで列挙**、他章は MUTED でグレーアウト。右下にページ番号。
4. **内容ページ**: 右上ナビ（現在章 TEAL）。GOLD「N.M タイトル」＋GOLD罫線。中央に図/本文。任意でコールアウト箱。左下に出典（MUTED 9.5pt「N. Author Year」）、右下に「M/総数」。
5. **ナビ帯**: 右上に**8 個**のホームプレート型（下向き五角形）タブを横一列。各タブ＝上に章番号（小）＋下に短章名。現在章＝TEAL塗り＋白文字（やや強調）、他＝`D9D9D9`塗り＋MUTED文字。右端は本文領域右端に揃え、上端 y≈0.15in。8個入るよう1タブ幅を PPG(7個)より少し狭める。
6. **全幅罫線**: タイトル直下の水平線。GOLD `BF9000`、太さ約3pt、左 x≈0.5in〜右 x≈12.8in。
7. **コールアウト箱**: 角丸長方形、塗り `F2E2B3`、細い GOLD 枠、中の文字は INK か GOLD。要点を1行で。

## 図版スタイル（matplotlib で作成 → PNG 透過で貼付）
- 太めの黒/濃紺の主曲線、補助は青 `2E6FBF`・橙 `E8853B`・赤 `C00000`・グレー破線。
- 塗りは淡色（青 `DAE3F3`、橙 `FCE4D6`）。注記は赤の引き出し線＋赤ドット。
- 軸は最小限（必要な軸のみ、時間軸は右向き矢印＋「時間」等）。フォントは図内も Meiryo 相当（sans）。
- 白背景で書き出し、pptx 側で白を透過にするか、図の背景を透明で保存（`savefig(transparent=True)`）。
