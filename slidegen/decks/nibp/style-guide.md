# 執筆ガイド（Opus 用）— 医学本文の書き方と出力形式

## 対象と狙い
- 読者: **麻酔科医・同僚**（踏み込んだ内容）。用語はそのまま（MAP, oscillometric, Korotkoff, peripheral amplification, CPB 等）。
- スライドは「見て話す」講義用。ブレットは**要点のみ短く**、深い説明は notes（スピーカーノート）へ。

## 絶対ルール（ユーザーCLAUDE.md 準拠）
1. **臨床的に正確に。** 麻酔科医が誤りを即座に見抜く。標準ガイドライン・生理学に当てる。
2. **著作権セーフ＝完全オリジナル文。** ガイドライン本文・教科書・過去スライドの丸写し禁止。論点は参照してよいが文章・数値表現は自作。
3. **安全クリティカルな数値は裏取り必須。** 下記の量的主張は WebSearch / PubMed 等で権威ある出典に照合し、確度を `sources` と `confidence` に残す。数値に自信がなければ notes に「要確認」と明記（捏造しない）。
   - カフ幅 ≈ 上腕周囲径の 40%、ブラダー長 ≥ 80%（AHA 推奨）
   - 静水圧補正 ≈ 0.75 mmHg/cm（血柱の高さ差）
   - 自動血圧計の検定 AAMI/ISO 81060-2：平均差 ≤ 5 mmHg・SD ≤ 8 mmHg
   - 術中低血圧の閾値 MAP < 65 mmHg（および SBP 低下 % 基準）と AKI・心筋障害の関連
   - オシロメトリック法は MAP を最も正確に測る（SBP/DBP は推定）
4. 正答・数値は一意に。曖昧なら範囲＋出典を示す。

## ブレットの書式
- 各ブレット ≤ 全角28字程度、体言止め/簡潔に。1スライド 3〜6 ブレット。
- 過度な階層化を避ける（第2階層まで）。記号は「・」。数値・単位は半角。

## 図の仕様（📊スライド）
`figure` に、Sonnet が matplotlib で正確に描けるだけの指示を書く：
- `kind`: waveform | schematic | comparison | flow | table
- `spec`: 軸・曲線・注記・近似値を文章で具体的に（例:「横軸=カフ圧 mmHg 高→低、縦軸=拍動振幅。振幅は釣鐘状に増減し、最大振幅点=MAP。SBPは最大振幅の約55%立ち上がり点、DBPは約85%下降点、を破線注記」）。
- `labels`: 図中に入れる日本語/英語ラベルの配列。
- デザインは design-spec.md の図版スタイルに従う（色・淡塗り・赤注記）。

## 出力形式（重要）
担当章ごとに **JSON ファイルを1つ書き出す**: `~/Desktop/nibp-slides/content/chNN.json`（NN=01..08、ゼロ埋め）。
スキーマ:
```json
{
  "chapter": 1,
  "chapter_title_ja": "歴史",
  "nav_short": "歴史",
  "slides": [
    {
      "id": "1.1",
      "title": "…（GOLD大タイトルになる。ID番号は含めず本文のみ）",
      "bullets": ["…", "…"],
      "callout": "…（任意。1行の要点。不要なら null）",
      "notes": "…（スピーカーノート。根拠・数値の背景・話す内容）",
      "figure": { "kind":"…", "spec":"…", "labels":["…"] },   // 不要なら null
      "sources": ["Author Year, 誌名/ガイドライン名", "…"],
      "confidence": "high|medium|low（量的主張の裏取り度）"
    }
  ]
}
```
- スライドID・枚数は `outline.md` に厳密に従う（増減しない）。
- `sources` は出典名のみ（フッター表示用の短い形、例「Pickering 2005, AHA Sci Stmt」）。本文への長文引用はしない。
- JSON は妥当（末尾カンマ無し・UTF-8）に。書き出したら短い要約だけ返す（本文は返さない）。
