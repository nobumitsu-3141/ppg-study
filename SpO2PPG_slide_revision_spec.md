対象ファイル: `SpO2PPG________4.15.pptx`（全 47 枚）
作成日: 2026-07-18
検査ツール: `slide-format` スキル同梱 `slide_lint.py`（+ 独自の座標・フォント解析）

> **この文書の位置づけ**
> 本デックは PPG 系の川副式デックで、組版機構（decklib.py・build スクリプト・テンプレ）は
> Mac ローカルにある想定のため、**実際の再組版は Mac の Claude Code で行う**運用です。
> Web 環境にはリフロー（自動再配置）エンジンが無く、フォントを機械的に 22pt へ上げるだけでは
> 枠固定のまま「あふれ・重なり」が増えて悪化します。
> そこで本文書は **① 何を直すか（欠陥リスト）＋ ② どう直すか（本文リライト・サイズ指定）** を
> スライド単位でまとめた改定仕様です。Mac の build にそのまま流し込めます。

  かつ **各タイトルは 21cm 手前で収まる文言に短縮**（G-3 の該当題）。タイトルとチップ間に

  - 現デザインを残す場合は「既存の川副式デックの意匠」として lint 例外扱いとし、この文書に
