# 反射波位置の数式的解析と反射指数 RI ― 術中後負荷指標としての提案

> SpO₂（PPG）波形の反射波を「前進波＋反射波」に分解し、その**位置（時間）**と**大きさ（高さ）**
> を数式的に定義したうえで、大きさ＝反射指数 **RI（reflection index）** を**術中後負荷（afterload）の
> トレンド指標**として用いる、という作業仮説をまとめる。本稿は確立事実の要約と、未検証の提案とを
> 明確に区別して記述する。プレゼン資料 `SpO2PPG_4.16.pptx` の第4章に節 **4.4〜4.6** として反映した。

---

## 0. 要旨

- 末梢脈波 `P(t)` は、左室駆出による**前進波 W_f** と、末梢からの**反射波 W_r** の重ね合わせとして表せる。
- 主反射点を1つ仮定すると反射波は前進波の遅延・縮尺コピー `W_r(t) ≈ Γ·W_f(t − τ)` で近似でき、往復時間 `τ ≈ 2d / PWV`。
- 合成波の**前進波ピーク P1** と**反射波ピーク P2** の時間差 `ΔT ≈ τ` が「位置」、振幅比 `P2/P1 ≈ Γ` が「大きさ」を与える。
  - 位置：`SI = 身長 / ΔT ∝ PWV` ―― 大血管スティフネス（**時間軸**）。
  - 大きさ：`RI = P2 / P1 ≈ Γ` ―― 反射の大きさ（**高さ軸**）。
- 反射係数 `Γ = (Z_L − Z_c)/(Z_L + Z_c)`。末梢トーヌス・**SVR** が上がると `Z_L↑ → Γ↑ → P2↑ → RI↑`。
- **提案**：`ΔRI`（麻酔導入前ベースラインからの変化）を、後負荷の反射・トーヌス成分の**非侵襲・連続・校正不要**なトレンド指標として追う。位置 `ΔT/SI` で半ば慢性の大血管スティフネスを補正する。
- **限界（作業仮説として明示）**：RI は本来「安静時・半ば慢性」の指標として検証されており、1拍ごとの急性変動追跡は未検証。振幅比ゆえ**心拍数（HR）の交絡**を受けやすい。指尖反射は局所か全身かが曖昧。**前向きに RI と SVR/Ea を対応づける検証が必要**。

## 1. 背景 ―― 反射波の「位置」と「大きさ」

動脈波形は、左室駆出により末梢へ進む**前進波（incident / forward wave）**と、血管分岐・末梢抵抗・
インピーダンス不整合の点で反射して中枢へ戻る**反射波（reflected wave）**の合成として理解される。
指尖で記録される波形の収縮期ピークは主に前進波を、その後に続く拡張期側のピーク／変曲点（重複切痕を
含む）は主に反射波の帰還を映す。したがって「波形から血管を読む」ことの核心は、**反射波の大きさ（高さ）
とタイミング（位置）を読むこと**に帰着する（本リポジトリ `SpO2_PPG_waveform_analysis_flow.md` §2）。

本稿は、この「位置」と「大きさ」を数式として明示し、そのうち**大きさ（RI）を後負荷側の指標**として
推せるかを検討する。

---

## 2. 反射波位置の数式化

### 2.1 前進波＋反射波の重ね合わせモデル

末梢脈波を次で近似する（主反射点を1つと仮定した最小モデル）。

```
P(t) = W_f(t) + W_r(t)
W_r(t) ≈ Γ · W_f(t − τ)
τ ≈ 2d / PWV
```

- `W_f`：前進波、`W_r`：反射波。
- `Γ`：反射係数（反射の大きさ、0〜1）。
- `τ`：反射点までの往復時間。`d` は反射点までの実効距離、`PWV` は脈波伝播速度。

反射波は前進波の**遅延（τ）**・**縮尺（Γ）**されたコピーとして立ち上がる。実際には分散（波形の鈍り）を
伴うが、位置と大きさの定義には上式で十分である。

### 2.2 位置 ―― ΔT と SI（時間軸）

合成波の前進波ピーク `P1` と反射波ピーク `P2` の時間差を `ΔT` とすると `ΔT ≈ τ`。これを身長で規格化した

```
SI = 身長 / ΔT   ( ∝ PWV )
```

が**スティフネス指数**であり、頸動脈–大腿動脈 PWV と相関して大血管スティフネスを反映する
（Millasseau 2002）。**位置は主に大血管スティフネス（伝播の速さ）で決まる時間軸の量**である。

### 2.3 大きさ ―― RI と反射係数 Γ（高さ軸）

反射波ピーク高を前進波ピーク高で割った比が**反射指数**である。

```
RI = P2 / P1   ( ≈ Γ )
```

RI は反射の大きさ、すなわち小〜中動脈の特性・小血管トーヌスを反映する（Chowienczyk 1999、Millasseau 2002）。
硝酸薬投与で反射波が減じて RI・変曲点位置が下がることが観察されており、「変曲点位置＝末梢トーヌス」という
発想の直接の起点である。**大きさは主に反射係数（末梢インピーダンス）で決まる高さ軸の量**である。

> **位置と大きさは別の軸である。** `ΔT`（＝いつ帰るか）は大血管スティフネス／PWV、`RI`（＝どれだけ帰るか）は
> 反射係数／末梢トーヌスを映す。両者を併読することで、後負荷の「反射・トーヌス成分」と「スティフネス成分」を
> 切り分けられる。

### 2.4 単峰性化への対応（SDPPG）

加齢・高スティフネスでは反射波が早期帰還して収縮期に融合し、波形が一峰性化して肉眼的に P2 が消える。
このとき二次微分 **SDPPG（加速度脈波）**が潜在的な変曲点を極値として顕在化させる：c 波が末梢反射（後期
収縮期の再上昇）、d 波が収縮期末、e 波が重複切痕に対応する。したがって生波形で P2 が見えなくても、
反射波の位置情報は c〜e 波として残り、P2 を操作的に同定できる（Takazawa 1998）。

## 3. 反射係数 → 末梢インピーダンス → 後負荷

反射は血管インピーダンスの不整合点で生じる。境界での反射係数は伝送線路・インピーダンス整合の一般論から

```
Γ = (Z_L − Z_c) / (Z_L + Z_c)
```

- `Z_c`：大血管の**特性インピーダンス**（壁スティフネス・径で規定）。
- `Z_L`：末梢の**負荷インピーダンス**（細動脈トーヌス、ほぼ SVR に対応）。

末梢トーヌス・**SVR が上がると `Z_L↑ → Γ↑ → 反射波 P2↑ → RI↑`**、血管拡張では逆に低下する。これは
本リポジトリおよびプレゼン資料の記述と整合する：

| 事象 | 波形・指標の動き | 主な根拠 |
|---|---|---|
| SVR・血管トーヌス ↑ | 反射波・切痕が「高さ」方向に上昇、RI ↑ | Awad 2007（SVR と相関するのは振幅より切痕を含む幅、r=0.56） |
| 麻酔中の血管収縮／拡張 | 切痕位置×振幅の6分類が SBP・SVR・コンプライアンスと相関 | Tusman 2019 |
| 末梢トーヌス低下（硝酸薬） | 反射波が減じ RI・変曲点位置が下降 | Chowienczyk 1999 |

> **用語の注意。** ここでの「後負荷成分」は SVR（定常）と波動反射（拍動）を指す。実効動脈エラスタンス Ea
> （後負荷の集約指標 ≈ 収縮末期圧/1回拍出量）や、壁スティフネスそのもの（PWV に反映）とは区別する。RI が
> 直接映すのは Ea ではなく、反射の大きさ（≒末梢トーヌス／SVR 寄りの反射・拍動成分）である。

## 4. 提案 ―― RI を術中後負荷トレンド指標に

後負荷は **定常成分（SVR）** と **拍動成分（Ea・波動反射）** からなる。RI（= P2/P1 ≈ Γ）はこのうち
反射・トーヌス成分を**一拍ごとに**映す形態指標である。以下を提案する。

1. **ΔRI を追う。** 絶対値ではなく、麻酔導入前ベースラインからの変化 `ΔRI` を後負荷トレンドとして追跡する
   （本プロジェクトの設計原則：校正不要・個人内の Δ を追う）。
2. **SpO₂ 一点から。** 追加モニタを要さず、非侵襲・連続に取得できる。
3. **位置で補正。** 位置 `ΔT/SI`（半ば慢性の大血管スティフネス）で交絡を補正し、急性の `ΔRI` を抽出する。
4. **AGC 耐性。** RI は**同一拍内の振幅比**であり、臨床モニタの自動ゲイン制御（AGC）が絶対振幅を破壊しても
   比としては保存されやすい。生振幅（高さ・面積）より AGC に強い点は、実機での実装可能性を高める。

期待する読み：**昇圧・血管収縮 → RI↑ ／ 血管拡張・深麻酔 → RI↓**。

## 5. 限界と検証設計（正直な但し書き）

| **作業仮説である。** 以下は本稿の提案であり、確立事実ではない。断定を避け、検証設計を併記する。 |
| --- |

- **時間スケールの不一致。** SI・RI は本来、安静時被験者で**半ば慢性**の血管特性を評価する指標として検証されて
  おり、1拍ごとの急性 SVR／前負荷変動をリアルタイムに追う用途は未検証である
  （`SpO2_PPG_waveform_analysis_flow.md` §5.2 の注意）。術中の急性トーヌス変化には、従来は切痕位置・幅や
  SDPPG の d/a が対応しやすいとされてきた。
- **HR 交絡。** 振幅比である RI は心拍数の交絡を受けやすく、時間指標である SI のほうが交絡に強い
  （`perioperative_stiffness_outcomes.md`：系統的レビュー）。周術期のように HR が大きく動く状況では、
  HR 補正（例：AIx@75 に倣う正規化）と SI 併用が要る。
- **局所か全身か。** 指尖の反射が局所トーヌスを見ているのか全身 SVR を見ているのかは曖昧で、指尖 PPG⇔SVR の
  相関は中等度にとどまる（Middleton 2011、r=0.54）。多部位・相対変化での解釈が要る。

**検証設計（反証可能な形）。** 全身麻酔症例で、①昇圧薬・血管拡張薬・輸液負荷の各介入時に `ΔRI` が SVR/Ea の
変化と同方向・用量依存に動くか、②HR 補正後に相関が残るか、③位置 `ΔT/SI` 補正で急性成分の分離が改善するか、
を前向きに検証する。侵襲的 SVR（熱希釈）・Ea を参照基準とする。

**第7章（esCCO 精度改善案）との接続。** RI を含む PPG 由来の血管指標は、PWTT ベースの心拍出量推定 esCCO の
較正係数 K を後負荷側から補正する枠組み（本リポジトリ `esCCO_K_correction_hypothesis.md`）の一要素にもなりうる。
